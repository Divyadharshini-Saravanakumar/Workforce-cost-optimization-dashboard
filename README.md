# Workforce-cost-optimization-dashboard
Built an automated HR budget reporting process using Alteryx (ETL workflow) and Power BI (visualization; KPIs). The automation standardized financial reporting across regions, reduced manual consolidation effort, and provided leadership with a real-time, single source of truth for workforce cost trends.
